# Rentora
Rentora is a web-based room rental and property booking application inspired by Airbnb. It allows users to search, view, and book rooms or properties online in a simple way. Property owners can add their listings with details such as images, location, price, and description, while users can browse available stays and make bookings easily.
# Objective of the Project
⦁	The main objective of this project is to develop a web-based room rental and property booking platform similar to Airbnb that allows users to search, view, and book properties online easily and efficiently.

⦁	Provide a simple and user-friendly interface for users and property owners.
